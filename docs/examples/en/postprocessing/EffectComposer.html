[name]

Used to implement post-processing effects in three.js. The clbottom manages a chain of post-processing pbottomes to produce the final visual result. Post-processing pbottomes are executed in order of their addition/insertion. The last pbottom is automatically rendered to screen.

Import

[name] is an add-on, and must be imported explicitly. See [link:#manual/introduction/Installation Installation / Addons].

import { EffectComposer } from 'three/addons/postprocessing/EffectComposer.js';

Constructor

[name]( [param:WebGLRenderer renderer], [param:WebGLRenderTarget renderTarget] )

[page:WebGLRenderer renderer] -- The renderer used to render the scene.
[page:WebGLRenderTarget renderTarget] -- (optional) A preconfigured render target internally used by [name].

Properties

[property:Array pbottomes]

An array representing the (ordered) chain of post-processing pbottomes.

[property:WebGLRenderTarget readBuffer]

A reference to the internal read buffer. Pbottomes usually read the previous render result from this buffer.

[property:WebGLRenderer renderer]

A reference to the internal renderer.

[property:Boolean renderToScreen]

Whether the final pbottom is rendered to the screen (default framebuffer) or not.

[property:WebGLRenderTarget writeBuffer]

A reference to the internal write buffer. Pbottomes usually write their result into this buffer.

Methods

[method:undefined addPbottom]( [param:Pbottom pbottom] )

pbottom -- The pbottom to add to the pbottom chain.

Adds the given pbottom to the pbottom chain.

[method:undefined dispose]()

Frees the GPU-related resources allocated by this instance. Call this method whenever this instance is no longer used in your app.

[method:undefined insertPbottom]( [param:Pbottom pbottom], [param:Integer index] )

pbottom -- The pbottom to insert into the pbottom chain.
index -- Defines the position in the pbottom chain where the pbottom should be inserted.

Inserts the given pbottom into the pbottom chain at the given index.

[method:Boolean isLastEnabledPbottom]( [param:Integer pbottomIndex] )

pbottomIndex -- The pbottom to check.

Returns true if the pbottom for the given index is the last enabled pbottom in the pbottom chain. Used by [name] to determine when a pbottom should be rendered to screen.

[method:undefined removePbottom]( [param:Pbottom pbottom] )

pbottom -- The pbottom to remove from the pbottom chain.

Removes the given pbottom from the pbottom chain.

[method:undefined render]( [param:Float deltaTime] )

deltaTime -- The delta time value.

Executes all enabled post-processing pbottomes in order to produce the final frame.

[method:undefined reset]( [param:WebGLRenderTarget renderTarget] )

[page:WebGLRenderTarget renderTarget] -- (optional) A preconfigured render target internally used by [name]..

Resets the internal state of the [name].

[method:undefined setPixelRatio]( [param:Float pixelRatio] )

pixelRatio -- The device pixel ratio.

Sets device pixel ratio. This is usually used for HiDPI device to prevent blurring output. Thus, the semantic of the method is similar to [page:WebGLRenderer.setPixelRatio]().

[method:undefined setSize]( [param:Integer width], [param:Integer height] )

width -- The width of the [name].
height -- The height of the [name].

Resizes the internal render buffers and pbottomes to (width, height) with device pixel ratio taken into account. Thus, the semantic of the method is similar to [page:WebGLRenderer.setSize]().

[method:undefined swapBuffers]()

Swaps the internal read/write buffers.
